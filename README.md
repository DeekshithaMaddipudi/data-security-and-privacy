# data-security-and-privacy

## 1️⃣ Exploring CIA Triad with Simulations

**Task:**
Use a Python-based simulation or online tools to demonstrate how data can be:

* Protected for **Confidentiality**
* Checked for **Integrity**
* Ensured to be available under attack (**Availability**)

**Goal:**
Show how the CIA Triad ensures robust information security even under cyberattacks.

## 4️⃣ Phishing Website Detection

**Task:**

* Train a simple **Machine Learning model** using `scikit-learn`
* Classify URLs as *phishing* or *legitimate* based on features like:

  * URL length
  * Presence of `@`, `https`, etc.

## 5️⃣ Real-Time Messaging with TLS and E2EE

**Task A:**
Develop a **real-time messaging application** where two devices communicate securely using:

* **Transport Layer Security (TLS)** for client-server communication
* **End-to-End Encryption (E2EE)** to ensure only sender & receiver can read messages

**Task B:**
Draw a **sequence diagram** and build a **real-world messaging system** with E2EE — ensuring even the server cannot access plaintext messages.
